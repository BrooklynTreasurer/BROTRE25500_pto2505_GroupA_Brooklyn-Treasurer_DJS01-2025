## ğŸ“ Project Description  

The **DJS01 Vanilla JavaScript Podcast App** is a responsive web application that allows users to **browse podcast shows** on a landing page and **view detailed information in a modal**, without navigating away from the page.

Users can:

- View podcast show previews displayed dynamically on the landing page  
- Click a podcast to open a **modal** with detailed show information  
- Explore show descriptions, genres, seasons, and episode counts  
- Close the modal easily and return to browsing  
- Experience a responsive UI based on provided wireframes  

This project highlights skills in **DOM manipulation, modular JavaScript, object-oriented programming, functional programming principles, and UI/UX design**.
